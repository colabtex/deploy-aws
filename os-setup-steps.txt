1. From "Ubuntu Software", search for "Visual Studio Code" and download the program (just called "code")
2. Remove all favorites except VS Code, Firefox, Nautilus, and Trash
3. Open new terminal/shell
4. Switch to sudo using $ { su - }
5. Make user sudo by running the following: $ { usermod -aG sudo your_username }
6. So that the updates are applied run the following (and after rebooted, log back in as your_username) $ { sudo -r shutdown now }
7. Install updates. To do so, if necessary, by running the following (and entering your password): $ { sudo apt update }
8. Install updates by running $ { sudo apt upgrade -y }
9. From the terminal, install git by running $ { sudo apt install git-all -y }
